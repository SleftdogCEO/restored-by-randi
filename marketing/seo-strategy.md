## On-Page Optimization Checklist

### Every Page

- [ ] Unique title tag (50–60 characters) including primary keyword
- [ ] Meta description (150–160 characters) with keyword + compelling CTA
- [ ] H1 tag with primary keyword (one per page)
- [ ] H2/H3 subheadings using related keywords
- [ ] URL slug is short, descriptive, keyword-rich (e.g., `/mastectomy-bras`)
- [ ] Internal links to 2–3 related pages/products
- [ ] Image alt text describing the image with natural keyword use
- [ ] Schema markup (Product, BreadcrumbList, Organization, FAQ)
- [ ] Mobile-friendly layout (test with Google Mobile-Friendly Tool)
- [ ] Page load speed < 3 seconds (compress images, lazy load)

### Product Pages

- [ ] Unique product descriptions (never copy manufacturer text)
- [ ] Product schema markup (name, price, availability, reviews)
- [ ] Customer reviews section (schema-enabled for rich snippets)
- [ ] "You may also like" related products section
- [ ] FAQ section at bottom (target "People Also Ask" queries)
- [ ] Size guide link
- [ ] Insurance coverage callout

### Collection/Category Pages

- [ ] 150–300 word intro paragraph with keywords
- [ ] Filterable product grid
- [ ] Breadcrumb navigation
- [ ] Internal links to blog posts about that category

### Blog Posts

- [ ] 1,000–2,000 words minimum for pillar content
- [ ] Target one primary keyword + 3–5 related keywords
- [ ] Include images with alt text
- [ ] Internal links to product pages (naturally)
- [ ] External links to authoritative sources (ACS, Susan G. Komen)
- [ ] Table of contents for posts over 1,500 words
- [ ] Author bio with E-E-A-T signals (cancer survivor, credentials)

## Local SEO (Even for E-Commerce)

Even as an online store, local SEO signals build trust and help with E-E-A-T.

### Setup

- [ ] Google Business Profile (see setup guide)
- [ ] Consistent NAP (Name, Address, Phone) across all listings
- [ ] Local business schema on website
- [ ] Yelp listing
- [ ] BBB listing
- [ ] Facebook business page with same NAP
- [ ] Apple Maps listing

### Why It Matters for E-Commerce

- Google Business Profile appears in branded searches
- Reviews on Google build trust signals
- Local listings create backlinks
- "Near me" searches can capture local customers
- Builds E-E-A-T (Expertise, Experience, Authority, Trust)

## Technical SEO Checklist

- [ ] XML sitemap submitted to Google Search Console
- [ ] Robots.txt configured correctly
- [ ] SSL certificate active (HTTPS everywhere)
- [ ] 301 redirects for any changed URLs
- [ ] No duplicate content (canonical tags where needed)
- [ ] Structured data validated (Google Rich Results Test)
- [ ] Core Web Vitals passing (LCP < 2.5s, FID < 100ms, CLS < 0.1)
- [ ] Mobile-first design verified
- [ ] 404 page with helpful navigation
- [ ] Breadcrumb navigation on all pages
- [ ] Clean URL structure (`/category/product-name`)
- [ ] Image compression (WebP format preferred)
- [ ] Lazy loading for below-fold images
