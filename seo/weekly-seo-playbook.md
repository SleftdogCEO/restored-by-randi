# Weekly SEO Playbook

Cadence aligned to Sleft weekly rhythm: Monday publish, Wednesday optimize, Friday distribute + review.

## Weekly Rhythm

### Monday - Publish and Index
- [ ] Publish 1 new high-intent blog post
- [ ] Add FAQ section and schema to post
- [ ] Add 2-3 internal links to money pages
- [ ] Submit new URL to Google Search Console and Bing

### Wednesday - Optimize Existing Assets
- [ ] Refresh 1-2 existing posts with updated FAQs
- [ ] Improve title and meta description for CTR
- [ ] Add or improve Related Reads block
- [ ] Validate canonical and breadcrumb schema on edited pages

### Friday - Distribution and Technical Review
- [ ] Review GSC queries, impressions, CTR, and avg position
- [ ] Check coverage/indexing errors and fix blockers
- [ ] Validate sitemap freshness and robots directives
- [ ] Log wins, losses, and next week priorities

## Monthly Rhythm

- [ ] Publish 4 high-intent posts
- [ ] Optimize 4 existing posts
- [ ] Create or improve 1 local landing page section
- [ ] Run 1 technical SEO audit pass (schema + internal links + crawlability)

## KPI Targets

| Metric | Target |
|--------|--------|
| Organic Clicks | +20% month over month |
| Impressions | +25% month over month |
| Avg CTR | +0.5 to +1.0 points over 60 days |
| Money Page Entrances | +15% month over month |
| Virtual Fitting Quiz Starts | +10% month over month |
