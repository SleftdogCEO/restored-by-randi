# Schema Conventions

Aligned to Sleft technical checklist conventions.

## Required by Template

### Homepage
- Organization
- WebSite
- SearchAction

### Blog Index
- BreadcrumbList

### Blog Post
- BreadcrumbList
- Article
- FAQPage (when FAQ section exists)

### Money and Local Pages
- WebPage
- FAQPage (when FAQ section exists)
- MedicalBusiness/LocalBusiness on local intent pages

## Article Schema Fields Standard

- author
- datePublished
- dateModified
- mainEntityOfPage
- image
- publisher (Organization + logo)

## Validation Standard

- Validate with Google Rich Results Test
- Validate with Schema.org validator
- Re-check after template changes
